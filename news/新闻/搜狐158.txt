原标题：是时候将面容识别带到 MacBook 上了
如果您是一位 Mac 用户，可能感觉到了苹果的「偏心」——iPhone 和 iPad 往往会先获得新功能，Face ID 可以说是一个真实的例子。苹果甚至已经开始尝试使用结合「苹果登录」的 Touch ID 和 Face ID 来登录 iCloud 了，而 MacBook 至今没有 Face ID，iMac 至今没有 Touch ID。
无论怎样，苹果花了两年时间来来推广 Face ID，并宣传这是一种更安全便捷的登录方式。那么现在应该是时候让 MacBook 用户也获得这个特权了。
MacBook准备就绪
当苹果在 2017 年推出 Face ID 时，它作为安全又方便的解锁 iPhone 的方式备受关注。和竞争对手的面部识别系统通常依赖于脸部的简单平面图像不同，Face ID 使用点投影仪，泛光照明器和红外摄像头来创建用户脸部的 3D 扫描，大大降低出错率。相较于 Touch ID 大约有五分之一的机会出现误判，苹果表示使用 Face ID 的出错率将低至百万分之一。它也很方便，不管你留胡子还是戴了帽子都能识别，并且再不用被潮湿或油腻的手指所困扰。
将它带到 MacBook 也不会有问题。在 iPhone 上，Face ID 所需的各种相机和传感器占据了设备顶部一定空间，使得全面屏上开了一个口，也就是大家调侃的「刘海」，代表着苹果设计师对现有技术条件的妥协。
但是，在 MacBook 上，这不会是一个问题。由于前置摄像头的存在，MacBook 本来就很宽的屏幕边缘并不在乎多开一个放置深感相机和传感器系统的位置，当然，苹果可以缩小 MacBook 屏幕边框，只为 Face ID 保留一个刘海，不过鉴于当年 iPhone X 刘海引起了一系列吐槽，苹果可能会对此三思。
另外，使用 iPhone 时，必须将其按纵向方向握住，才能使 Face ID 工作，这可能有点烦人。在 MacBook 上这将不是问题。
苹果已经给新款 Mac 以及 MacBook 加上了 T2 安全芯片，为安全环境打下了很好的基础，因此不必为开发 Face ID 的新安全环境而烦恼，T2 早已铺好道路。
事实上，现在已经有至少一种使用 Face ID 登录 Mac 的方法。比如 Unlox 这样的第三方应用可以让你通过 iPhone 上的 Face ID 或 Touch ID 解锁 Mac，当然，这远远不如 Mac 上原生 Face ID 解锁那么方便，毕竟它要求你有另一个苹果产品才能工作。但事实上，第三方都能实现的功能苹果却迟迟未能动手，这暗示着实现这一功能时还需面对一些挑战。
转身迎接挑战
苹果 MacBook Air 和带 Touch Bar 的 MacBook Pro 目前都有 Touch ID 按钮，这是对于用户来说是一个福音。
如果用的是 Face ID，事情会变得更加容易，用户只需要看一下屏幕（反正使用 MacBook 时你也一直看着屏幕）就能提供同样快捷的解锁和其他身份认证。
近几年苹果一直在努力实现 Mac 端的新身份验证手段。这几年来的努力成果几乎都发生在 iPhone 上，2019 年，苹果将重心重新回归 Mac 产品线，我们迎来了 iMac Pro 和新款 Mac Pro，然后就是不久前刚刚进行了较大变动的 MacBook Air 和 MacBook Pro 升级，Mac 系列王者归来，这让我们重新燃起了新技术登陆 Mac 的希望。
有用户会顾虑 Face ID 将增加 MacBook 的成本，但根据目前的报道和苹果的动作来看，这家公司的目标很可能是彻底改造 MacBook，一款新的 16 英寸型号高端 MacBook Pro 显然正在研发中，苹果可以轻而易举地给这款划时代产品添加 Face ID，将其作为新的超级高端 MacBook 推向市场，并配之以高价。
增加 Face ID 所付出的成本对于苹果用户来说不是什么难以接受的事，少量价格升级便可换取出色的隐私保护，安全性和易用性，Face ID 肯定物有所值。来吧苹果，是时候展现真正的技术了。
